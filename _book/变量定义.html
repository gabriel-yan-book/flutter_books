变量定义

变量定义

一、通过 var 来进行定义

   main() {
     var num = 100;
     var str = 'hello 西阁'; //! 一旦声明，类型就确定了，确定了类型后就不能随意更改了
     // str = 100; //! 更改类型就会报错
     print( str );
   }

二、通过 String 来进行定义

  main() {
    String str = 'Hello 西阁';
    // str = 100 ; 报错
    print( str );
  }

三、通过 int 来进行定义

  main() {
    int num = 100;
    print( num );
  }
