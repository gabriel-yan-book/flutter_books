Set

1. Set是什么？

Set是没有顺序且不能重复的集合

2. Set的作用是什么？

Set主要作用是去除列表中重复元素

3. Set如何使用？

  void main() {
    var s = new Set();
    List list = ['🐻','⚽','🎂','🏈','abc','abcd','🐻','⚽','🎂','🏈'];
    s.addAll( list );
    print( s ); //{'🐻','⚽','🎂','🏈','abc','abcd'}
  }
