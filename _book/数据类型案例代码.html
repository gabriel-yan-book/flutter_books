数据类型案例代码

数据类型案例代码

一、Number

  1. int 整型
    main() {
 int a = 10 ;
 int b = 10 ;
 print( a );
 print( b );
}

  2. double 既可以是整型，也可以是浮点型

    main() {
 int a = 10.3 ;
 int b = 10.3 ;
 print( a );
 print( b );
}

  3. 运算符

    // + - * / % 这些运算都是可以的
main() {
 int a = 10 ;
 int b = 10 ;
 int c = a + b ;
 print( c )
}

二、String

  1. 通过 var 可以进行字符串类型定义
    main() {
 var str = '西阁'; // 你可以使用单引号
 var str2 = "西阁"; // 你也可以是用双引号
 print( str );
 print( str2 );
}
  2. 通过 String 可以进行字符串类型定义
    main() {
 String str = '西阁'; // 你可以使用单引号
 String str2 = "西阁"; // 你也可以是用双引号
 print( str );
 print( str2 );
}
  3. 可以通过 ''' ''' 或者 """ """ 来进行多行字符串定义
    main() {
 String str = '''
   this is 西阁
   this is 西阁
   this is 西阁
 ''';
 String str2 = """
   this is 西阁
   this is 西阁
   this is 西阁
 """;
 print( str );
 print( str2 )
}
  4. 字符串拼接
    main() {
 String str = 'this is';
 String str2 = '西阁';
 print( str + str2 ); // 可以通过 + 来连接两个变量
 print( '你猜我是谁？$str $str2' ); // 也可以通过 $+变量名来进行字符串的连接 
}

三、Boolean

  1. 如何定义
    main() {
 bool f = true ; // 通过 bool 来对布尔值来进行定义
 bool f2 = false ;
 print( f );
 print( f2 );
}
  2. 如何使用布尔值进行逻辑判断
    main() {
 bool f = true;
 if ( f ) {
   print('条件成立');
 } else {
   print('条件不成立')
 }
}

四、List

  1. 字面量直接定义
    main() {
 var arr = [ 1,2,3,4 ];
 print( arr );
 print( arr.length );// 列表的长度
 print( arr[0] ); // 我们可以通过下标来进行列表中每一个元素的访问
}
  2. 通过 List 方法来进行定义
    main() {
 var arr = List();
 arr.add(1);
 print( arr );
}
  3. 定义列表的类型
    main() {
 var arr = List<String>();// 表示定义了一个元素类型为字符的列表
 arr.add('a');
 print( arr );
}

五、Map 字典

  1. 直接定义
    main() {
 var map = {
   'id': 1,
   'name': '西阁',
   'age': 18
 };
 print( map );
 print( map['id'] ); // 我们可以通过这种方式来访问字典中的每一个键值对
}
  2. 可以通过 new Map() 来进行定义
    main() {
 var map = new Map();
 map['id'] = 1;
 map['name'] = '西阁';
 map['age'] = 18;
 map['sex'] = 'man';
 print( map );

六、通过 is 来进行数据类型的判断

  1. 案例
    main() {
 var a = 1000;
 if ( a is String ) {
   print('a是字符串类型');
 } else if ( a is int ) {
   print('a是数值类型');
 } else {
   print('a是其他数据类型');
 }
}
